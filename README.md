# MATLAB-Project-1

An applied linear algebra project modeling a network of routers as a system of linear equations, solved using multiple methods to analyze data flow and generate network optimization recommendations.

## About

- Modeled a router network as a system of linear equations and solved it using four methods: row reduction (Gaussian elimination), LU decomposition, matrix inverses, and Cramer's Rule, comparing results across all approaches.
- Analyzed the solved system to interpret data flow rates across network nodes, identifying bottlenecks and formulating informed optimization recommendations.
- Applied MATLAB as a computational tool for matrix operations, demonstrating practical use of linear algebra in a real-world network engineering context.
